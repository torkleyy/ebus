Software driver for [eBUS] (energy bus) written in Rust.

[eBUS]: https://ebus-wiki.org/lib/exe/fetch.php/ebus/spec_prot_12_v1_3_1_e.pdf

* `no-std`
* zero dependencies (except optional `log`)
